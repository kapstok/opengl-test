# OpenGL 4.0 Test

Based on [Glitter](https://github.com/Polytonic/Glitter).

## Cloning & Building

Use the `--recursive` argument when cloning.

```bash
cd build

# UNIX Makefile
cmake ..

# Mac OSX
cmake -G "Xcode" ..

# Microsoft Windows
cmake -G "Visual Studio 14" ..
cmake -G "Visual Studio 14 Win64" ..
...
```
